# 📖 Manual de Operação: Sistema Contratos

Este guia descreve como utilizar as principais funcionalidades do sistema **Contratos**.

---

## 🔑 1. Acesso ao Sistema
O sistema é acessado exclusivamente através do **Portal GestorGov**.
- Clique no ícone "Módulo de Contratos" no Portal.
- Você será autenticado automaticamente e redirecionado para a **Dashboard**.

## 📊 2. Dashboard Administrativo
A Dashboard exibe os KPIs em tempo real:
- **Resumo Financeiro:** Valor total em contratos vigentes.
- **Alertas de Vencimento:** Notificações automáticas de contratos que expiram nos próximos 30, 60 e 90 dias.
- **Atividade Recente:** Últimas ações realizadas no sistema.

## 🏢 3. Gestão de Fornecedores (Prestadores)
- **Cadastrar Fornecedor:** Acesse o menu "Fornecedores" e preencha os dados cadastrais (CNPJ, Endereço e Contatos).
- **Contatos Múltiplos:** É possível cadastrar diversos contatos para o mesmo fornecedor (ex: Comercial, Financeiro).

## 📄 4. Gestão de Contratos
- **Listagem Principal:** Use os filtros por Diretoria, Ano e Fornecedor.
- **Dossiê do Contrato:** Clique em um contrato para visualizar todos os detalhes, aditivos e histórico de anexos.
- **Inserir Novo Contrato:** Preencha os dados principais e vincule ao fornecedor cadastrado.
- **Upload de Anexos:** Formatos suportados (PDF, JPG, PNG, DOCX).

## ⚙️ 5. Configurações Administrativas (Admin)
- **Tabelas Auxiliares:** Gerencie as opções de Diretorias, Fontes de Recursos, Modalidades e Categorias.
- **Permissões de Usuários:** Controle quem pode visualizar ou editar registros no sistema.
- **Acesso Global:** É possível permitir leitura global para todos os usuários do Portal através de uma chave nas configurações.
